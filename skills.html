Job Roles & Skills Set

My set of specialization

★ System Engineer :

• Installations and configurations of Computer Server (Red Hat Linux, Centos, Ubuntu, Windows)
• Install and manage services/daemons/Apps (web services, database services, storage services, mail services, VoIP services, and more).
• Manage user or group policy (local policy, Active Directory, LDAP, Kerberos, and more).
• Monitor, Maintenance, updates, and patches for the services.
• Virtualization (Openstack, KVM, Qemu, VMWare vSphere, Docker, Kubernetes, and more).

★ Cloud Engineer :

• Public Cloud Providers (Google, AWS, Alibaba, More).
• Cloud Infrastructure Design.
• Create and Manage Cloud Resource.
• Setup and Configuration Services.
• Scaling and Automation Services.
• Logging, Monitoring, and Observability.

★ Big Data Engineer :

• Installations and configurations of Big Data Servers (Hadoop Ecosystem).

• Monitor Server cluster resource usage.

★ Network Engineer :

• Devices Brand (Mikrotik, Ubiquiti, Fortinet, more).

• Firewall (Protocol, Port, Services, Network, L7 filtering).
• Domain Name Pointing.

★ IT Support Engineer :

• Helpdesk support.
• Application support.
• Remote support (Remote Desktop, Radmin, other online apps).
• Networking Support (Cable, WIFi, Routing check, etc).
